# PM

PM 관련 문서와 대시보드 HTML을 모아두는 폴더입니다. 브라우저에서 HTML 파일을 바로 열어 PM 업무 흐름, 우선순위 판단, 리포트화 방향을 확인할 수 있습니다.

## 포함 파일

- `PM 통합대시보드.html`: 개발/QA/릴리즈 현황을 한 화면에서 보는 통합 대시보드 예시
- `PM 업무 방향성.html`: 개발 PM 역할과 업무 방향성을 정리한 인터랙티브 문서
- `PM 판단 대시보드(심플형).html`: 요청 입력, 자동 점수, Now/Next/Hold 보드를 중심으로 한 간단한 버전
- `PM 판단 대시보드(실무구성형).html`: Jira/CSV 입력, 자동 점수, PM 확정, Backlog, 공유 요약, 팀 병목, 결정 로그까지 포함한 확장 버전

## 추천 사용 순서

1. `PM 판단 대시보드(심플형).html`로 기본 개념을 빠르게 확인합니다.
2. `PM 판단 대시보드(실무구성형).html`로 실제 운영에 가까운 흐름을 보여줍니다.
3. 필요에 따라 `PM 통합대시보드.html`과 `PM 업무 방향성.html`을 함께 참고합니다.

## 실무구성형 주요 흐름

- Jira JQL 또는 CSV 데이터로 요청을 가져옵니다.
- 자동 점수 탭에서 영향도, 긴급도, 가치, 리스크, 난이도, 공수 기준으로 추천 레인을 확인합니다.
- PM 확정 탭에서는 모든 요청이 Backlog에서 시작합니다.
- PM이 카드 안의 Now, Next, Hold 버튼을 직접 눌러 확정합니다.
- 확정 결과는 공유용 요약, 담당 팀별 병목, 결정 로그에 반영됩니다.
