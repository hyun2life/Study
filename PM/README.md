# PM

`PM` 폴더는 PM 관련 문서를 정리하고, 업무 방향이나 대시보드 형태를 HTML로 시안화해 보는 공간입니다.

현재는 읽기 쉬운 단일 HTML 문서 형태로 정리되어 있어 브라우저에서 바로 열어 확인할 수 있습니다.

## 포함 파일

- `PM 통합대시보드.html`: PM 현황을 한눈에 보기 위한 대시보드 시안
- `PM 업무 방향성.html`: PM 역할과 업무 방향을 정리한 문서

## 용도

- PM 업무 구조 정리
- 대시보드 UI 시안 실험
- HTML 문서 형태의 자료 정리와 공유

## 사용 방법

- 각 HTML 파일을 브라우저에서 열어 바로 확인합니다.
- 필요하면 내용을 수정해 문서형 페이지나 시안으로 계속 확장할 수 있습니다.

## 메모

- 코드보다는 문서와 시각화 시안에 가까운 폴더입니다.
- 이후 PM 템플릿, 체크리스트, 회의 정리 문서 등을 함께 쌓아가기에 좋은 구조입니다.
