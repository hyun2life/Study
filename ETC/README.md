# ETC

`ETC` 폴더는 메인 프로젝트 바깥에서 따로 보관하고 싶은 결과물이나 실험용 작업을 모아두는 공간입니다.

성격이 조금 다른 프로젝트를 분리해 두는 용도로 쓰고 있고, 현재는 이모티콘 후처리 프로젝트가 들어 있습니다.

## 하위 폴더

- `A mere dad`: 완성된 이모티콘을 PNG로 제작하고 다듬는 프로젝트

## 용도

- 메인 작업과 분리해서 보관할 프로젝트 정리
- 결과물 중심 폴더 보관
- 실험용 또는 보조 성격의 작업 아카이브

## 메모

- 하위 프로젝트마다 별도 `README.md`를 두고 관리하는 방식이 잘 맞는 폴더입니다.
- 프로젝트가 늘어나면 같은 방식으로 하위 폴더를 추가해 정리하면 됩니다.
