Live Page Feedback

현재 탭의 화면 위에 피드백 수집 도구를 활성화합니다. 활성화 후 원하는 요소를 클릭하여 피드백 마크다운을 만드세요.

  1. 모드 활성화: 위의 [피드백 모드 켜기] 버튼을 클릭합니다.
  2. 요소 선택: 페이지에서 검토할 요소 위에 마우스를 올리고 클릭합니다.
  3. 메모 작성: 텍스트 입력창에 필요한 수정 의견을 적고 [피드백 복사]를 누릅니다.
  4. 전달: 클립보드에 복사된 마크다운을 에이전트 채팅창에 붙여넣습니다.
💡 Pro Tips:
• 작성 중 Esc를 누르면 선택 모드가 즉시 종료됩니다.
• 요소 클릭 시 텍스트 내용이 자동으로 수집되어 편리합니다.
