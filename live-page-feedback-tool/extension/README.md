# Live Page Feedback Chrome Extension (Agentation 연동형)

## 프로젝트 목적
실제 운영, 스테이징, 또는 로컬 개발 환경(`http://localhost:3000` 등)의 임의의 웹페이지 상에서 마우스 클릭만으로 화면 요소의 정밀한 정보(CSS Selector, 뷰포트 크기, 마우스 클릭 좌표, 바운딩 박스, 주변 텍스트 등)를 수집하여 AI 에이전트에 프롬프트용으로 전달할 수 있는 구조화된 피드백을 만들어내는 Chrome Extension(크롬 확장 프로그램)입니다.

이 확장 프로그램은 오리지널 오픈소스 **[benjitaylor/agentation](https://github.com/benjitaylor/agentation)** 라이브러리를 React 런타임과 함께 완전히 내장(Bundle)하여 주입하는 구조로 설계되어, 라이브 사이트의 보안 제한(CORS, X-Frame-Options, CSP)을 무력화하고 오리지널 Agentation의 강력한 UI 검토 기능들을 그대로 활용할 수 있습니다.

---

## 주요 기능
1. **피드백 모드 토글**: 확장 프로그램 팝업의 "피드백 모드 켜기/끄기" 버튼으로 간편하게 Agentation 툴바를 활성화하거나 웹페이지에서 완전히 삭제(Unmount)할 수 있습니다.
2. **다양한 어노테이션 도구 지원 (Agentation 제공)**:
   - **요소 선택**: 마우스 오버로 대상 요소를 하이라이트하고 클릭하여 즉시 피드백을 기록합니다.
   - **텍스트 선택**: 화면 위 텍스트를 드래그하여 정확히 지칭하고 의견을 남길 수 있습니다.
   - **영역 선택**: 특정 범위를 마우스 드래그 박스로 선택해 시각적인 피드백을 전달합니다.
   - **애니메이션 일시정지**: 동적인 전환 효과나 애니메이션이 있는 순간의 UI 상태를 포착할 수 있습니다.
3. **마크다운 클립보드 복사**: 툴바 내에 내장된 복사 버튼을 눌러 에이전트에 바로 전달할 수 있는 마크다운 프롬프트를 복사합니다.

## 폴더 구조
```text
live-page-feedback-dist.zip (배포 압축파일)
└── extension/
    ├── manifest.json   # Chrome Extension 설정 명세 (Manifest V3)
    ├── popup.html      # 확장 프로그램 아이콘 클릭 시 뜨는 팝업 UI
    ├── popup.js        # 팝업 UI의 동작 및 content.js 동적 주입 제어
    ├── content.js      # [Vite 빌드 생성물] React + Agentation 컴포넌트 번들 스크립트
    ├── icon.png        # 확장 프로그램 전용 로고 아이콘
    ├── run-feedback-chrome.bat # [팀원 전용] 원클릭 확장 자동 탑재 크롬 실행기
    └── README.md       # 사용자 매뉴얼 및 프로젝트 가이드 (본 문서)
```

## 🚀 팀원용 원클릭 실행 방법 (가장 추천)
팀원들은 번거로운 설치나 크롬 설정 조작 필요 없이 다음 순서로 원클릭 기동이 가능합니다.

1. 배포받은 **`live-page-feedback-dist.zip`** 파일의 압축을 해제합니다.
2. 해제된 폴더 내의 **`run-feedback-chrome.bat`** 파일을 더블클릭하여 실행합니다.
3. 화면에 뜨는 창에서 피드백을 남길 **대상 웹페이지 주소**를 입력하고 엔터를 누릅니다.
   - 주소를 적지 않고 엔터를 치면 기본값인 로컬 대시보드(`http://localhost:3000`)로 접속됩니다.
4. **자동으로 피드백 확장이 탑재된 상태의 크롬 창**이 띄워지며 웹페이지가 열립니다.
5. 크롬 우측 상단의 확장 프로그램 아이콘을 눌러 **Live Page Feedback**을 켜고 사용하시면 됩니다.

## 🛠 개발자용 수동 설치 및 실행 방법
사용자 정의 빌드 작업이 필요한 개발자용 가이드라인입니다.

### Step 1: content.js 빌드 및 zip 패키징
1. `agentation-demo` 폴더로 이동합니다.
2. **`pack-extension.bat`** 스크립트를 실행합니다.
   - `content.js` 빌드가 누락되어 있으면 자동으로 Vite 빌드를 우선 실행한 뒤, 빌드 완료된 `extension` 폴더 전체를 배포용 `live-page-feedback-dist.zip` 파일로 자동 패키징합니다.

### Step 2: 크롬 브라우저에 확장 프로그램 정적 등록 (자동 런타임을 안 쓸 경우)
1. Chrome 브라우저의 `chrome://extensions` 에 접속합니다.
2. 우측 상단의 **'개발자 모드'**를 활성화합니다.
3. 좌측 상단의 **'압축해제된 확장 프로그램을 로드합니다.'** 버튼을 누릅니다.
4. 본 프로젝트의 `agentation-demo/extension` 폴더를 로드하여 등록합니다.

## 주의사항
- **브라우저 보호 페이지**: 크롬 정책상 브라우저의 기본 설정창(`chrome://`), 크롬 웹스토어 등의 특수 도메인에서는 보안을 위해 외부 스크립트 주입이 차단되므로 작동하지 않습니다. 일반 웹사이트에서 작동해 주세요.
