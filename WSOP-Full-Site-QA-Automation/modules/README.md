# Modules

WSOP 전체 사이트 자동화는 기능별 모듈로 나누어 진행합니다.

각 모듈은 아래 기준을 갖습니다.

- 독립 실행 가능해야 한다.
- 입력 설정과 출력 산출물이 명확해야 한다.
- 실패 항목은 재현 가능한 URL, 증거 파일, 원인 분류를 남겨야 한다.
- 최종 통합 리포트에 합칠 수 있는 JSON 결과를 제공해야 한다.
