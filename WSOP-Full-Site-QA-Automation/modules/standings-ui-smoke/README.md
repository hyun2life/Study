# Standings UI Smoke

WSOP 전체 사이트 자동화의 2단계 후보 모듈입니다.

## 목적

Player Standings 화면의 기본 사용자 흐름이 깨지지 않았는지 빠르게 검증합니다.

## 후보 테스트

- standings 페이지가 정상 로드된다.
- 기본 카테고리 목록이 표시된다.
- 주요 카테고리 이동 시 URL 또는 콘텐츠가 바뀐다.
- 각 카테고리에 최소 1개 이상의 선수 row가 표시된다.
- 선수명 링크를 클릭하면 Player Profile로 이동한다.
- standings row의 핵심 컬럼이 비어 있지 않다.
- 페이지 이동 또는 더 보기 동작 후 row가 중복 없이 추가된다.
- 빈 데이터 또는 네트워크 오류 상태가 화면에서 깨지지 않는다.
