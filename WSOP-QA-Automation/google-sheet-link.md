# WSOP QA Automation TC Matrix

Google Sheet:

https://docs.google.com/spreadsheets/d/1YVeyjdoLetBq8yU67C5YTs6fpLyFKn1MUZhlxqB2WL8/edit?usp=drivesdk

## Sheets

- `TC_Master`: 운영 기준 TC 마스터
- `Test_Run`: 테스트 실행 회차 관리
- `Execution_Log`: TC별 실행 결과 기록
- `Defect_Log`: 결함 기록
- `QA_Report`: Run ID 기준 QA 결과 자동 집계
- `TC_List`: 대분류/중분류/소분류 기반 TC 목록
- `Category_Filter`: 자동화 후보 필터 기준
- `Phase_Backlog`: 단계별 자동화 백로그
- `Legend`: 판정 기준과 도구 분류
